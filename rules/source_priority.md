# Source Priority

情報源の優先順位は以下とする。

1. 公式サイト
2. 公式PDF・料金表・約款
3. 運営会社のプレスリリース
4. 官公庁・公的機関
5. 信頼できる比較メディア
6. 個人ブログ・SNS・口コミ

料金、キャンペーン、契約条件、端末代、解約金、通信制限は必ず公式情報を優先する。

公式情報が確認できない場合は、本文では断定せず「公式サイトでは確認できませんでした」と記載する。

検索API、検索結果スニペット、AI要約、競合記事の要約は、情報源を発見するための補助として扱う。本文で事実を断定する根拠には、公式サイト、公式PDF、一次情報ページ、または信頼できる公的資料の直接確認を使う。

## 競合記事に記載された基本情報の扱い

競合記事が商品・サービスの基本情報表として掲載している項目は、自記事の比較表・個別紹介で扱う候補として必ず抽出する。

抽出対象の例:

- 公式URL
- 運営会社
- 料金体系
- 入会金・教材費・分割払い
- 受講期間
- 受講形式
- 学習内容・カリキュラム
- 取扱AIツール
- 講師・メンター
- 質問サポート
- コミュニティ
- 転職・副業・案件獲得サポート
- 補助金・給付金対応
- 受講者数・卒業生実績
- 無料相談・無料体験
- 返金保証・解約条件
- 注意点・デメリット

ただし、競合記事に記載があるだけでは事実確定しない。公式サイトや公式PDFで同じ情報を確認できた場合のみ、公式情報を根拠として本文・比較表に使う。

公式サイトに記載が見つからず、競合記事にだけ記載がある場合は、以下を区別して記録する。

- 情報源種別: `競合記事`
- 競合記事URL
- 掲載箇所または該当見出し
- 記載内容
- 公式確認状況: `公式未確認`
- 本文利用可否: `断定不可。目視確認・公式再確認が必要`

この扱いの情報は、本文では断定せず、必要に応じて「競合記事では記載があるが、公式サイトでは確認できませんでした」といった確認状況を明記する。
