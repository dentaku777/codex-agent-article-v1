# Reference Article Analysis Prompt

あなたはSEO比較記事の参考記事分析担当です。

`brief.md` と `xx.memo/url.md` に記載された参考記事を読み、同種の記事を汎用的に再現するための構成・表現・画像・CTAの型を分析してください。

作業前に必ず以下のルールを確認してください。

- `rules/reference_article_policy.md`
- `rules/output_format.md`

出力先：
`articles/<slug>/reference_article_analysis.md`

分析項目：

- 記事テーマ
- 想定検索意図
- ファーストビュー構成
- h2/h3構成
- 導入文の型
- 比較表の種類
- ランキングの見せ方
- 個別商品・サービス説明の型
- 利用者アンケート・口コミの使い方
- 外部リンクの使い方
- 画像・比較画像・図解・CTA画像の種類
- CTA配置
- FAQ構成
- 記事の独自性
- 自記事に取り入れるべき汎用パターン
- そのまま模倣してはいけない表現・構成

注意：

- 参考記事は構成・表現パターンの参考であり、本文を流用しない
- 料金、条件、ランキング順位などの事実は参考記事から確定しない
- 参考記事内の画像は、種類・役割・構図を分析し、画像自体は複製しない
