## 3. 深掘りリサーチで最低限抽出する項目

各スクールの公式情報リサーチでは、少なくとも以下を抽出する。

| 項目 | 抽出方針 |
|---|---|
| 公式URL | LP、本体サイト、料金ページ、FAQ、特商法、約款、キャンペーンページを分けて記録 |
| 運営会社 | 会社名、設立年、所在地、関連する運営実績を公式情報で確認 |
| 料金体系 | 入会金、受講料、月額、分割、教材費、キャンペーン、補助金適用後価格を区別 |
| 受講期間 | 最短期間、標準期間、継続型か買い切り型かを確認 |
| 受講形式 | オンライン、通学、動画、ライブ、マンツーマン、チャット、コミュニティの有無 |
| 取扱AIツール | ChatGPT、Claude、Gemini、画像生成、動画生成、Dify、n8nなど、公式に明示された範囲で記録 |
| 補助金対応 | リスキリング、専門実践教育訓練、一般教育訓練給付金などの対応可否を公式情報で確認 |
| 主要差別化 | 公式情報、競合記事、アンケートから比較軸として使える特徴を整理 |
| 累計受講者・実績 | 数値、表示箇所、確認日、出典URLを記録。出典不明なら未確認扱い |
| 注意点 | 料金条件、解約条件、最低契約期間、申込停止、キャンペーン期限、根拠不足を明記 |

料金、キャンペーン、契約条件、解約条件、スペック、累計実績は公式サイトを最優先し、出典URLのない事実は断定しない。

- 同一ドメインの重複記事は除外する

- 検索上位5位の記事は優先度を高く扱う
