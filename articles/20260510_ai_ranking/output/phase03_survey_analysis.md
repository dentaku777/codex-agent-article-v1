## 1. 分析の位置づけ

現時点のアンケート回答は25件であり、最終予定の約300件に対してサンプルが少ない。

そのため、本フェーズの分析結果は以下の用途に限定して使う。

- 記事構成の仮説づくり
- 読者ニーズの把握
- 比較軸・評価軸の候補化
- FAQ候補の抽出
- 口コミ・不満・注意点の方向性確認
- フェーズ5の公式情報リサーチで重点確認する項目の整理

一方で、以下には単独で使わない。

- ランキング順位の確定
- 評価点の確定
- スクールごとの優劣の断定
- 受講者全体の傾向としての断定

本文で使う場合は、「アンケート初期回答25件では」「現時点の集計では」など、暫定値であることを明記する。最終300件集計後に、比率・代表コメント・満足度平均を差し替える。

## 14. 自由回答から見える読者インサイト

### 14-1. ポジティブ要素

- 独学では断片的だった知識を体系的に学べた。
- 実務で使えるプロンプト設計を学べた。
- 文書作成、資料作成、議事録要約、Excel関数作成など、現職での業務効率化に直結した。
- 最新ツールや生成AIの使い方を幅広く学べた。
- 動画＋チャット、マンツーマン、メンター相談が学習継続に役立った。
- 補助金を使えた場合、費用負担を抑えて学べた。

### 14-2. ネガティブ要素

- 受講しただけで副業案件が取れるわけではない。
- 料金が高いと感じる回答が複数ある。
- モチベーション維持が難しい。
- 実務にどう落とし込むかは、自分で試す必要がある。
- コミュニティやサポートが期待より弱い場合がある。
- 教材の更新頻度、最新AI動向への追随が不安材料になる。

### 14-3. 受講前に確認すべきこと

- どの業務・目的で生成AIを使いたいか。
- 副業、転職、業務効率化のどれを重視するか。
- 学習時間を確保できるか。
- 補助金・給付金の対象条件と手続き。
- 追加費用の有無。
- カリキュラムの更新頻度。
- サポート範囲、返信体制、卒業後サポート。
- 副業案件紹介の実態と支援範囲。
- 無料相談・体験動画・口コミで、自分に合うか。

## 15. 記事構成への反映案

### 15-1. リード・結論で使う要素

- 初期回答では、受講目的は「業務効率化」と「副業」が多い。
- 生成AIスクールは、単にツールを覚える場ではなく、実務で使うプロンプト設計や活用パターンを体系的に学ぶ場として訴求する。
- ただし、副業案件獲得や収入化はスクールだけで完結しない点を注意書きとして明示する。

### 15-2. 選び方に入れるべき見出し候補

1. 目的に合うカリキュラムかを確認する
2. 実践課題で業務に落とし込めるかを確認する
3. サポート・質問体制が継続しやすいかを確認する
4. 補助金・給付金の対象条件を確認する
5. 副業・転職サポートは実績と範囲まで確認する

### 15-3. 比較表に入れるべき列候補

| 列候補 | 理由 |
|---|---|
| 料金総額 | 費用満足度が低く、不満も多い |
| 補助金・給付金対応 | 活用成功・対象外・手続き断念が分かれている |
| 受講期間 | 1〜6ヶ月が中心 |
| 受講形式 | 動画＋チャット、ライブ、マンツーマンの違いが重要 |
| 学べるAIツール | 最新ツール網羅が満足点で最多 |
| 実践課題の有無 | 業務効率化実感と直結 |
| メンター・質問サポート | 継続しやすさと不満回避に重要 |
| コミュニティ | 不満点にも出ている |
| 副業・転職支援 | 期待と現実のギャップがある |
| 卒業後の教材閲覧 | 満足点として一定数ある |

| 補助金・給付金は誰でも使えますか？ | 補助金活用状況 | 高 | 対象者・対象講座・申請条件があるため公式/公的情報で回答 |
| 仕事が忙しくても続けられますか？ | 学習継続の不満、社会人属性 | 高 | 受講形式、質問体制、学習時間、サポートで判断 |
| 受講前に何を準備すべきですか？ | 自由回答 | 中 | 目的明確化、学習時間、無料相談、補助金条件確認を提示 |
| どのAIツールを学べるスクールを選ぶべきですか？ | 受講後用途、満足点 | 中 | 文章、画像、動画、資料作成、業務自動化に分ける |

## 17. フェーズ5への引き継ぎ

アンケート傾向から、公式情報リサーチで重点確認すべき項目は以下。

| 確認項目 | 理由 |
|---|---|
| 料金総額、入会金、教材費、分割払い | 費用不満が大きい |
| 補助金・給付金の対象条件 | 活用状況が分かれ、手続き断念もある |
| 実践課題・課題添削の有無 | 実践スキル・業務効率化が主要ニーズ |
| 学べるAIツール | 最新ツール網羅が満足点で最多 |
| 質問サポート・メンター対応 | 継続しやすさと不満回避に重要 |
| コミュニティの有無・活動内容 | 不満点として出ている |
| 副業・転職支援の範囲 | 期待外れを防ぐため、支援内容を厳密に確認 |
| 卒業後の教材閲覧・サポート | 満足点として一定数あり |
| カリキュラム更新頻度 | 最新AI動向への追随不安がある |

## 18. フェーズ3の結論

25件時点では、生成AIスクールの読者ニーズは「現職での業務効率化」と「副業への応用」が中心に見える。

一方で、不満点としては「学習継続の難しさ」「費用の高さ」「副業案件獲得への期待外れ」が目立つ。したがって、本記事では単純なおすすめランキングだけでなく、目的別の選び方、費用対効果、補助金、サポート体制、副業支援の実態を丁寧に比較する必要がある。

最終的な300件集計後は、本ファイルの数値、比率、代表コメント、評価軸への反映度を更新する。
