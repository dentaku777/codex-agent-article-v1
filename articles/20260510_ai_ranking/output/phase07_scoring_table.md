## 1. 採点方針

本採点は、フェーズ6で確定した5軸・各20点、総合100点満点で行う。

```text
総合評価 = カリキュラム内容 + コースの種類 + 受講料（コスパ） + サポート体制 + 運営実績
```

### 採点時のルール

- 公式サイト、公式FAQ、公式料金ページ、特定商取引法ページ、公的機関情報で確認できる項目を加点根拠にする。
- 競合記事・口コミサイト・検索結果スニペットだけの情報は、掲載候補や再確認項目として扱い、加点根拠にはしない。
- 料金、補助金、契約条件、解約条件、返金保証、実績数値は公式確認できない限り断定しない。
- 公式情報が不足しているサービスは、該当軸の点数を抑え、本文では「公式確認できた範囲」と「未確認項目」を明記する。
- ユーザー確認済みの正規ランキングTOP10を深掘り対象として扱う。
- 公式で新規申込停止が確認されたサービスは、読者が現在申し込めるランキング本体から除外し、注意付き掲載に回す。

## 5. 実質月額・料金比較の計算方針

生成AIスクールは、月額制、買い切り型、短期集中型、補助金対象講座が混在するため、本文では以下の2種類で比較する。

### 5-1. 総額比較

```text
総額 = 受講料 + 入学金 + 教材費 + 必須システム利用料 + その他必須費用
```

### 5-2. 実質負担額

```text
実質負担額 = 公式確認済み総額 - 公式確認済み給付・補助見込み額
```

### 5-3. 月額制・サブスク型の比較

```text
比較用総額 = 月額料金 × 比較月数 + 入学金 + 教材費 + その他必須費用
```

- 契約期間が公式確認できる場合は、その契約期間を比較月数にする。
- 契約期間がない場合は、比較条件として3か月、6か月、12か月を別表にする。
- 補助金や助成金は、公式または公的情報で確認できるものだけ反映する。
- 法人向け助成金は、個人向け給付金と分けて表記する。
