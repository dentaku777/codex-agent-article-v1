## 1. 本ランキングの目的と基本方針

本ガイドラインは、「生成AIスクール おすすめ」を主キーワードとするランキング記事において、生成AIスクール・講座を公平に比較するための選定基準および評価方法を定めたものです。

生成AIスクールは、月額制の学び放題サービス、買い切り型教材、マンツーマン指導、通学型講座、補助金対象講座など形態が大きく異なります。また、受講者の目的も「業務効率化」「副業」「リスキリング」「転職」「AI開発」などに分かれます。

ただし、目的別の向き不向きは読者ニーズによって強弱が変わるため、本ランキングの点数軸には含めません。目的別おすすめは、記事本文内の目的別セクションで別途整理します。

本ランキングでは、生成AIスクールを総合比較するうえで共通して確認すべき以下の5軸を採点対象とします。

**基本方針**

- 公式サイト、公式FAQ、公式PDF、特定商取引法ページ、公的機関情報を一次情報として優先する
- 競合記事・比較メディアの記載は参考情報に留め、公式確認できない情報は加点根拠にしない
- 受講料は税込価格で統一し、入学金・教材費・システム利用料・分割条件も確認できる範囲で含める
- 補助金・給付金は、公式サイトまたは公的機関で確認できる場合のみ評価に反映する
- 取扱AIツール、カリキュラム更新、実践課題、サポート体制、運営実績を総合的に評価する
- 新規申込停止、料金非公開、公式情報不足、生成AI講座としての実態が薄い場合は減点またはランキング本体から除外する

## 3. 評価項目と配点

総合評価は以下の5軸各20点、合計100点満点で算出します。各軸は均等配点とし、特定の軸に偏重しない設計とします。

| 評価軸 | 配点 | 評価の観点 |
|:---|:---:|:---|
| (1) カリキュラム内容 | 20点 | 取扱AIツールの網羅性 / 最新領域カバー / プロンプト・画像・動画・業務自動化・AI開発への対応 / 教材更新頻度 / 教材の体系性 |
| (2) コースの種類 | 20点 | コース数・プラン数 / 対応領域（業務効率化・副業・マーケ・デザイン・動画・AI開発等）/ 対応レベル / 受講形式の柔軟性 |
| (3) 受講料（コスパ） | 20点 | 受講料の絶対額 / 月額制・買い切り・分割 / 入学金・教材費 / 補助金・給付金対応 / 返金保証 / 実質負担額 |
| (4) サポート体制 | 20点 | メンタリング・質問対応・コミュニティ / AIチューター / 課題添削 / 副業・転職支援 / 教材閲覧期限 / 卒業後サポート |
| (5) 運営実績 | 20点 | 運営会社の信頼性 / スクール運営歴 / 累計受講者数・卒業生数 / 法人研修実績 / 公的認定 / メディア掲載・第三者評価 |
| **総合評価** | **100点** | **5軸の合計** |

```text
総合評価 = カリキュラム内容 + コースの種類 + 受講料（コスパ） + サポート体制 + 運営実績
```

| スコア | 採点基準詳細 |

**生成AIスクール固有の補足**

補助金・給付金は、個人向けの教育訓練給付金、経産省系リスキリング支援、法人向けの人材開発支援助成金などが混在しやすい項目です。個人向け・法人向けを分け、公式確認できる制度だけを評価に反映します。

**料金計算ルール**

単発・期間固定講座:

```text
総額 = 受講料 + 入学金 + 教材費 + 必須システム利用料 + その他必須費用
```

月額制講座:

```text
比較用総額 = 月額料金 × 比較月数 + 入学金 + 教材費 + その他必須費用
```

補助金適用後:

```text
実質負担額 = 公式確認済み総額 - 公式確認済み給付・補助見込み額
```

注意:

- 税込・税抜を混在させない
- 公式に確認できない補助金・割引は計算に入れない
- 法人向け助成金は、個人向け補助金とは分けて扱う
- 期間限定キャンペーンは確認日を明記する

### (5) 運営実績（20点満点）

運営実績は、以下のサブ要素を総合して評価します。

生成AIスクールは市場形成から日が浅く、生成AIに特化した講座単体で10年以上の運営歴を持つケースは現実的に少ないため、ここでいう運営歴は「運営会社またはスクール事業全体の教育・指導実績」を指します。プログラミングスクール、ITスクール、資格スクール、法人研修などで培った指導ノウハウは加点対象にします。

一方で、生成AI講座としての開講実績、教材更新頻度、生成AIツール対応、法人向け生成AI研修の実績は、スクール全体の運営歴とは分けて確認します。スクール全体の運営歴が長くても、生成AI講座としての実態や更新性を公式情報で確認できない場合は、カリキュラム内容や運営実績で過度に加点しません。

|:---|:---:|:---|
| 運営歴・会社信頼性 | 5点 | 運営会社の設立年、スクール・教育事業全体の運営年数、特商法・会社概要の明確さ |

| メディア・第三者評価 | 3点 | メディア掲載、受賞、第三者比較サイト掲載、公式に確認できる評価 |

**減点対象**

- 特商法・運営会社情報が確認しにくい場合

## 5. 減点・対象外ルール

| 条件 | 評価処理 |
|:---|:---|
| 公式で新規申込停止が確認された | ランキング本体では対象外、または総合点から最大30点減点 |
| 料金が公式確認できない | 受講料（コスパ）は最大8点まで |
| 補助金が競合記事にしか記載されていない | 補助金加点なし |
| 生成AIカリキュラムの公式確認ができない | カリキュラム内容は最大8点まで |
| 講座マーケットプレイス・買い切り教材 | ランキング本体ではなく代替案枠。総合ランキング点は原則付けない |
| 法人向け助成金のみ確認 | 個人向け補助金としては加点しない。法人向け注記で扱う |
| 競合記事のみの実績数値 | 運営実績に加点しない。本文では `公式未確認` と明記 |
| 特商法・会社概要が確認できない | 運営実績を減点し、本文で注意点として扱う |

## 6. 同点時の順位決定ルール

総合点が同点の場合は、以下の順で上位を決定します。

| 優先順位 | 判定項目 | 理由 |
|:---:|:---|:---|
| 1 | 公式情報で確認できた項目数が多い | 読者が料金・条件を確認しやすく、情報の信頼性が高い |
| 2 | カリキュラム内容 + 実践性に近い要素が強い | アンケートでカリキュラム充実度・実践スキルが重視されていたため |
| 3 | 受講料（コスパ）の透明性が高い | 費用不満が多く、補助金・総額の確認が重要なため |
| 4 | サポート体制が手厚い | 学習継続の難しさがアンケートで最多の不満だったため |
| 5 | 運営実績が強い | 高額講座では運営会社の信頼性が重要なため |
| 6 | ユーザー指定TOP10の順位 | ユーザー指定順位を最終調整要素として尊重するため |
