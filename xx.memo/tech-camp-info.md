生成AI

ChatGPT
生成AIの中心的なツールを学び、AIの基礎力を身につける

Genspark
人間の代わりに複数のAIを使って、スライド作成、動画編集、旅行の計画立て、お店の予約などをこなす万能のAIエージェントです。

Cursor
AIがプログラミングを完全自動で書いてくれるCursorを学ぶことで、初心者でも自分だけのアプリが作れるようになります。

Dify
プログラミングスキルがなくても、Difyならコードを書かずにあなただけの業務効率化AIツールが作れるようになります。

Perplexity
今までの検索体験を超えるAI時代の検索ツールを使いこなす

ClaudeCode
Claude Codeを学ぶことで、AIがコード作成からテスト・実行までを代行し、作りたかったアプリや業務改善ツールを最短最速で形にする力が身につきます。

OpenClaw
OpenClawを学ぶことで、非エンジニアの方でも「自分専用のAIエージェント」を完成させることができます。

NotebookLM
AIが資料を読み解き整理してくれるNotebookLMを学ぶことで、大量の情報を効率よくまとめ、活用できる知的生産スキルが身につきます。


Webサイト制作

HTML
Webページの土台となるHTMLを学ぶことで、自分でサイトの構造を作るために必要なWeb制作の基本スキルが身につきます。

CSS
文字の色や配置、デザインの調整ができるCSSを学ぶことで、見た目が整った美しいWebページをつくる力が身につきます。

JavaScript基礎
Webページに動きを加えるJavaScriptの基礎を学ぶことで、ボタン操作やアニメーションなど、ワンランク上のサイトが作れるようになります。

jQuery
副業やWeb制作の現場でも使われることの多いjQueryを学び、短いコードで効率よく動きのあるWebサイトを作れるスキルが身につきます。

レスポンシブ対応
スマホやタブレットでも見やすいWebページを作るレスポンシブ対応を学ぶことで、あらゆるデバイスに対応できる制作力が身につきます。

AIアプリ開発

AIの基礎知識
AIがどうやって動いているのか、その仕組みや種類を学ぶことで、アプリ開発に必要な土台となる理解力が身につきます。

Python
AI開発でよく使われる人気のプログラミング言語Pythonの書き方を学ぶことで、自分の手でAIを動かせるプログラミングスキルが身につきます。

機械学習・深層学習
機械学習・深層学習という概念について学び、現代のAIがどのような仕組みによって高度な知能を実現しているのかを理解します。

自然言語処理
ChatGPTをはじめとする現代に大きな影響を与えている生成AIの根幹をなす、自然言語処理・大規模言語モデルなどの概念を学びます。

AIチャットボット開発
プログラミング言語Pythonを用いて、GPTモデルのAI機能を組み込んだ本格的なAIチャットボット開発をしていきます。

ノーコード開発
Dify、AzureML、Yoomなどのノーコードツールを使用して、コードを書かずにAIアプリを開発する方法を身につけます。

Webデザイン

デザインの基礎原則
デザインの基本的な原則や、配色・フォント・画像・配置のテクニックを身につけることで、見た人に伝わるクリエイティブを作る力を手に入れます。

Figma
現代のWebデザインの現場で多く使われているツールFigmaを学ぶことで、自分の手でデザインをカタチにするノウハウを身につけます。

UIデザイン
Webサイトを作る上での代表的なパーツや表現方法を学び、どんな人でも快適に使えるWebサイトの構築方法を学びます。

レイアウト設計
Webサイトの全体をデザインし、情報をわかりやすく整理するためのレイアウトの設計方法や、レイアウトを考える手順を身につけます。

Webデザイン実践
実際にテーマや要件に合わせて自分の手でWebデザインを作成することで、現場で通用する実践的なデザインスキルを身につけます。


業務効率化

GASの基礎知識
Google Workspaceを自動化するプログラミング言語、GASの基礎を学びます。変数、関数、条件分岐などの基本的な考え方から段階的に習得します。

表計算の自動化
スプレッドシートでの入力・更新・集計・書式設定などの自動化や、複数シート間のデータ連携や条件に基づいたデータ抽出を実現できるようになります。

ビジネス文書作成の自動化
定型文書の自動生成やテンプレートの活用、データに基づく文書作成などGoogle Docsを使用した業務効率化を学習します。

情報収集・検索の自動化
Web APIを活用した外部データの取得と活用方法を学び、取得したデータのスプレッドシートへの記録など、情報活用の基礎を身につけます。

スライド制作の効率化
スプレッドシートのデータを活用したスライドの自動生成を学び、グラフや表の埋め込み、テンプレートの活用など、効率的な資料作成を実現します。

スケジュール管理の効率化
Googleカレンダーとの連携による予定管理の自動化を学び、会議の自動登録や参加者への通知など、スケジュール関連業務の効率化を実現します。

#### AIエージェントコース
DAY1
11:00〜12:30
オリエンテーションとClaude Codeの導入
まずはAIエージェントの基本概念を整理し、業務効率化の全体像を確認します。その後、Claude Codeのインストールを行い、正常に動作する環境をセットアップ。AIに対してファイル作成や情報取得を命じ、実務パートナーとしての連携を開始します。

12:30〜13:30
休憩

13:30〜14:00
講義①：コンピュータとインターネットの基礎
AIへの指示（プロンプト）を正確に行うために必要な、コンピューターやインターネットの基礎知識を学びます。インターネットを介したデータ送受信のルールや、AIがPC内部でどのようなプロセスを経て操作を代行しているのか、その技術的背景を理解します。

14:00〜16:30
講義②：外部データの自動取得とCSVレポート生成
外部サービスから必要な情報を収集する手順を習得します。収集したデータをExcel等で即時活用できるCSV形式へ自動加工・保存する仕組みを構築。さらに、複数の処理を組み合わせて指定した時間にプログラムを自動実行させる設定を実践します。

16:30〜17:00
休憩

17:00〜19:30
講義③：Webアプリ形式の管理ツールの作成
情報の入力・保存・表示という基本機能を備えた「Todo管理アプリ」を、AIと共に制作します。要望を伝えて画面構成や機能を調整し、実用的なツールへと磨き上げる工程を体験。自身の構想がブラウザ上で即座に形になるスピード感を体感し、開発の可能性を広げます。

19:30〜20:30
ワークショップ：業務の棚卸しとAI導入プロセスの設計
1日目の学習を通じてAIの能力を把握した状態で、自身の業務を細かく分解します。AIに任せる領域と人間が確認すべき領域を定義し、導入後の運用フローを策定。「どの業務を自動化すれば最大の成果が得られるか」という、導入戦略を明確にします。

20:30〜21:00
初日の振り返り

DAY2

11:00〜14:00
講義④：自律拠点の構築とOpenClawの導入
クラウドサーバーを立ち上げ、24時間稼働するAIの専用拠点を構築します。安全な接続（SSH）と通信通路を設定し、強固なセキュリティ環境を整備。自律型AIエージェントである「OpenClaw」をインストールし、ブラウザ操作などをAIに完全委譲するためのセットアップを完遂します。

14:00〜14:30
休憩

14:30〜17:00
講義⑤：Gmail・LINE・Todoの自律連携実装

17:00〜17:30
休憩

17:30〜19:30
最終課題：オリジナル業務自動化ツールの開発
