#  「生成AIスクールおすすめランキング」 選定・評価基準書

発行日：2026 年 5 月 6 日<br>
発行元:各種通信講座 - 株式会社キャリア・ブレーン 比較・検証編集部

## 1. 本ランキングの目的と基本方針

本ガイドラインは、生成 AI スクール上位 10 校を対象としたランキング記事における選定基準および評価方法を定めたものです。

生成 AI 領域は 2023〜2025 年で急速に拡大し、社会人の副業・転職・業務効率化のためのスキル獲得手段として「生成 AI スクール」の需要が高まっています。受講料は月額 9,800 円から買切 80 万円超まで幅広く、リスキリング支援補助金（最大 70%）・専門実践教育訓練給付金（最大 80%）等の活用有無で実質負担額が大きく異なります。本ランキングでは、社会人受講生（25〜45 歳・副業/転職/業務効率化志向）の視点から、以下の基本方針に基づき評価を行いました。

**基本方針**

- 公式サイトに掲載された情報を一次情報として採用し、外部ソースは補助情報として扱う
- 受講料は税込価格で統一し、補助金・給付金適用後の実質負担も考慮する
- 取扱 AI ツール（ChatGPT / Claude / Gemini / Midjourney / Sora / Dify / RAG / LangChain / NotebookLM / Claude Code / n8n 等）の網羅性と最新領域（AI 駆動開発・AI エージェント等）への対応を重視する
- 運営実績は運営歴・累計受講者数・法人研修実績・公的認定（JDLA / GUGA）・第三者媒体掲載等の客観指標で評価する
- 事業継続性懸念（教室閉鎖・申込停止・集団訴訟報道等）は運営実績軸で減点対応する

## 2. 評価項目と配点

総合評価は以下の 5 軸各 20 点、合計 100 点満点で算出します。各軸は均等配点とし、特定の軸に偏重しない設計としています。

| 評価軸 | 配点 | 評価の観点 |
|:---|:---:|:---|
| (1) カリキュラム内容 | 20 点 | 取扱 AI ツールの網羅性 / 最新領域カバー（プロンプトエンジニアリング・AI 駆動開発・AI エージェント等）/ カリキュラム更新頻度 / 動画講義の体系性・教材のクオリティ |
| (2) コースの種類 | 20 点 | コース数・プラン数 / 対応領域（マーケ / エンジニア / デザイン / 動画 / 業務効率化 / 副業 等）/ 対応レベル / 受講形式の柔軟性（オンライン / 通学 / マンツーマン / 動画見放題）|
| (3) 受講料 (コスパ) | 20 点 | 受講料の絶対額 / 月額制 vs 一括 / 入会金・教材費 / 補助金・給付金対応（リスキリング 70% / 専門実践 80% / 一般教育訓練 20%）/ 無料体験・返金保証 / 実質負担額 |
| (4) サポート体制 | 20 点 | メンタリング・質問対応・コミュニティ / AI チューター等の独自サポート / 案件紹介・転職保証・副業支援 / 学習期間・教材閲覧期限 / 卒業後コミュニティ |
| (5) 運営実績 | 20 点 | 生成 AI スクール / 運営会社のスクール事業の運営歴 / 累計受講者数・卒業生数 / 法人研修・導入企業実績 / 公的認定（JDLA / GUGA）・補助金指定 / メディア掲載・第三者比較サイト評価・受賞歴 |
| **総合評価** | **100 点** | **5 軸の合計** |
